## Getting Started
```shell
git clone https://github.com/0x100Chefs/pomodoro-watch-backend-rust
cd pomodoro-watch-backend-rust
cargo run 
```

### Dependencies
The application is built on Docker, Rust, Redis and PostgreSQL, ensure to have the dependencies setup first
